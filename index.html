Welcome to SpectrumStylr!

What IS SpectrumStylr?

SpectrumStylr is a powerful (yet, simple to use) CSS and JavaScript Library. You can use this amazing tool to create rich pages without typing a single line of CSS code! Use SpectrumStylr to create marvelous sites, and attract users! After all, presentation is as important as accuracy.

Docs

But, how can I code with it? How do I know if this is the right class or ID? Our SpectrumStylr Docs will make sure you aren't left in the dark. Our Docs cover ALL aspects of SpectrumStylr — from buttons to links, from navbars to footers. In a nutshell, EVERYTHING!

Themes

Stay ahead with SpectrumStylr themes. Themes are bits of special code with certain default colors. They are simplified, but can beautify your web page. Plus, they're a change from the same old general code. They help you kill two birds with one stone.
